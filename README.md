## Portofolio-Version-4

### About This
Vue **SPA**
Responsive Protofilo, 

created by **Vue2** + Vue-Cli + Vuetify + Vue-Router,
and used **Axios** to get JSON data from public files.

---

### Tech Stacks
    - Vue2
    - Vue-cli
    - Vue-Router
    - Vuetify
    - Scss
    - Axios
    - JSON
    - vue-lazyload
    - vue-carousel
    - vue-typer
    - Material Design icons
