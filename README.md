## Portofolio-Version-4

### About This
Vue **SPA**
Responsive Protofilo, 

created by Vue-Cli3 + Vuetify + Vue-Router,
and used Axios to get data from JSON files.

### Tech Stacks
    - Vue2
    - Vue-cli3
    - Vue-Router
    - Vuetify
    - Scss
    - Material Design icons
    - JSON
    - Axios
    - vue-smooth-scroll
    - vue-typer
